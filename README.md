# 🍴 MHP – Most Happening Place

Welcome to **MHP**, a modern and responsive food ordering web app built using **Vite + React**. Designed to make your cravings just a click away!

---

## 🚀 Features

- 🍽️ Explore a dynamic menu
- 🛒 Add to cart and place orders
- 🔒 User authentication
- 🧾 View past orders
- 🎨 Sleek and responsive UI

---

## 🛠️ Tech Stack

- Frontend: **React** with **Vite**
- Styling: **CSS3**
- Backend: *(Coming Soon / Already Connected)*
- Payment: Stripe API (Test Mode)
